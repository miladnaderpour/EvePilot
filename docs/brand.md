## Meaning

- Orbit: autopilot, control, and navigation.
- Nodes: network topology.
- Motion lines: automation and speed.
- Wordmark: clean developer-tool identity.

## Typeface

Space Grotesk 600, lowercase, with tight tracking.

## Usage

Use the full logo for README headers and documentation pages.

Use the mark for GitHub avatars, compact UI areas, and places where the wordmark
would be too small.

Use the compact orbit-only favicon for browser and app icon sizes below 24 px.

Use dark variants on dark navy or black backgrounds.
